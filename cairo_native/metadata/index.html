§Code generation metadata

The metadata is used for various stuff that would be otherwise difficult or plain impossible with the current design, such as:

  • Pass compile-time constants that would otherwise have to be hardcoded in various places (ex. PrimeModuloMeta).
  • Declare FFI bindings to external libraries (ex. ReallocBindingsMeta).
  • Pass extra compilation info to the libfunc generators (ex. TailRecursionMeta).
